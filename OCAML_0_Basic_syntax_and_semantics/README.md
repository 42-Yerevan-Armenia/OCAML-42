# OCAML - 0 - Basic syntax and semantics

Short descriptions for exercises in this module (ex00..ex09):

- ex00: ft_test_sign — print "positive" or "negative" depending on an int (0 is positive).
- ex01: ft_countdown — print a countdown from n to 0, one number per line.
- ex02: ft_power — compute a^b for non-negative integers.
- ex03: ft_print_alphabet — print the lowercase alphabet on a single line.
- ex04: ft_print_comb — print all increasing 3-digit combinations (012..789) separated by ", ".
- ex05: ft_print_rev — print a string in reverse, followed by a newline.
- ex06: ft_string_all — return true if a predicate holds for every character of a string.
- ex07: ft_is_palindrome — return true if a string reads the same forwards and backwards.
- ex08: ft_rot_n — rotate alphabetical characters by n (preserve case; non-letters unchanged).
- ex09: ft_print_comb2 — print all unique ordered pairs from 00 to 99 (00 01, 00 02, ... 98 99).

Each exercise has its own `exNN/` directory and a single `.ml` file to submit.
