# OCAML - 0 - Pattern Matching and Data Types

Short descriptions for exercises in this module (ex00..ex08):

- ex00: `encode` — run-length encode a list into a list of `(count * element)` pairs.
- ex01: `crossover` — return common elements between two lists.
- ex02: `gray` — print the Gray code sequence of size `n`.
- ex03: `sequence` — generate the nth element of the look-and-say sequence as a string.
- ex04: `nucleotides` — define nucleotide types and `generate_nucleotide` from a char.
- ex05: `helix` — helix type, random helix generation, stringify and complementary helix.
- ex06: `rna` — convert a helix to messenger RNA (T -> U) and represent RNA type.
- ex07: `ribosome` — split RNA into triplets and decode into amino acids/proteins.
- ex08: `life` — full pipeline from helix generation to protein creation; display steps.

Each `exNN/` directory contains an implementation file you can compile with `ocamlopt`.
